Spotify Data Project

A look into a user's spotify listening history!

Overview

For this project, I analyzed a dataset containing a single spotify user's listening history. I used the built-in python libraries Pandas and MatplotLib to visualize the relationship between the amount of time the user would spend on each track and the reasons they started or ended listening to that track.

Key Features

  • Look into any user's listening history
  • Evaluate Spotify's recommendation algorithm
  • Visualize trends

How it Works

Visualizing MS Played vs. Reason Start:

The table was read using pandas, and MatplotLib was used to visualize a bar chart comparing the average milliseconds played of all the tracks for each starting reason.
